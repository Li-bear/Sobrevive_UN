Objetivo: Programa que ejecuta un juego que simula la vida universitaria
durante un semestre, donde las decisiones que tome el jugador afectaran en
su desempeño y supervivencia.

Información básica para ejecutar el programa:

- El programa necesita las siguientes librerías: turtle, tkinter y random.
  Ejecutar los siguientes comandos para instalar las librerías:
  
  pip install turtle
  pip install tkinter
  pip install random

- No hay que modificar ni eliminar el archivo records_usuario.

- El programa .py debe estar en la misma carpeta con las imágenes.
